## 기존 코드

```
private static int strike(String num, int[] arr, List<Integer> randomNum) {
        int count = 0;
        for (int i = 0; i < 3; i++) {
            if (num.charAt(i) - '0' == randomNum.get(i)) {
                count++;
                arr[randomNum.get(i)]--;
            }
        }
        return count;
    }
```
